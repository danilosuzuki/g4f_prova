Informações iniciais para configuração

  1. Abra o cmd (terminal) na pasta prova
  2. Certifique-se de ter o composer instalado na sua máquina. Caso não tenha baixe aqui
  3. Execute e cerifique-se de estar funcionando algum servidor web como Apache. Xampp é recomendado
  4. Rode o comando composer install esse processo pode demorar alguns minutos.
  5. Certifique-se que o laravel foi instalado com sucesso rodando o comando php artisan
  6. Caso o laravel esteja instalado, rode o comando php artisan migrate. Será perguntado se deseja criar a base de dados. Digite 'yes'.
  7. Após a base criada popule com o comando php artisan db:seed --class=Questao3Seeder. Caso deseje alterar a quantidade de itens a serem populados na base de dados altere no arquivo 'Questao3Seeder.php'. Como padrão são criados 10 séries, e são gerados 30 intervalos e atrelados aleatoriamente entre as séries geradas.
  8. Caso deseje rode o comando php artisan test para executar os testes das 3 questões
  9. O sistema se encontra pronto para uso. Rode o comando php artisan serve
